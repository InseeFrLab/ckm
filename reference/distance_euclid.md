# Euclidean distance

Calculates the Euclidean distance between original and perturbed count
vectors.

## Usage

``` r
distance_euclid(o, p)
```

## Arguments

- o:

  numeric vector. Original values

- p:

  numeric vector. Perturbed values

## Value

numeric. Euclidean distance value

## Examples

``` r
distance_euclid(1:100, 11:110)
#> [1] 100
```
