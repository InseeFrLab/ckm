# Sample dataset for CKM package

A synthetic microdata dataset containing 6,000 records with demographic
and geographic variables for testing purposes.

## Usage

``` r
dtest
```

## Format

data.table with 9 variables:

- SEXE:

  Gender (G = male, F = female)

- DIPLOME:

  Education level (5 categories)

- AGE:

  Age group (3 categories)

- DEP:

  Department code

- REG:

  Region code

- TYPE:

  Detailed category (5 levels)

- TYPE2:

  Aggregated category (2 levels)

- VAL:

  Random numeric value

- WEIGHT:

  Sampling weight
