# Mean absolute deviations

Calculates the mean absolute deviations between original and perturbed
values.

## Usage

``` r
mean_absolute_deviation(o, p)
```

## Arguments

- o:

  numeric vector. Original values

- p:

  numeric vector. Perturbed values

## Value

numeric. Mean absolute deviation

## Examples

``` r
mean_absolute_deviation(1:100, 11:110)
#> [1] 10
```
